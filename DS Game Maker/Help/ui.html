User Interface

Most commands in DS Game Maker can be achieved in one of two ways: by using the toolbar, or by using the menu bar (drop down commands from menus such as 'File', 'Edit' and 'Resources'). Here is an explanation of each command from the menu bar.

File Menu

  • New: Create a new game. This button actually launches a new instance of the DS Game Maker application.
  • Open: Open a saved game. This button presents a dialog for you to select a game file on your computer. The extension for DS Game Maker projects is .DSGM.
  • Open Last Project: After launching the program, use this button to load the project that was being edited last time you used the software.
  • Save: Saves the game under its current file name. If you are working with a new project and have not saved it yet, this button will perform the action of the 'Save As' button and let you save the file somewhere on your computer.
  • Save As: Saves the game under a different file name.
  • Test Game: Compiles your game and runs it in the included emulator.
  • Compile Game: Compiles your game and presents a choice for what to do next:
    • Play Game (runs it in the included emulator)
    • Save to Homebrew Kit (saves the game to your Homebrew Kit)
    • Save NDS File (lets you save the NDS file to be uploaded to a homebrew website or put on your flashcart)
    • Open Compile Folder (opens the folder containing the game source code)
  • Exit: Quits the program. If you have a project open, you will be asked if you would like to save any changes since the last save.

Edit Menu

  • Duplicate: Creates a copy of the currently open resource and opens the copy. You can also do this by right-clicking a resource in the tree at the left of the program and selecting 'Duplicate'
  • Delete: Deletes the currently open resource.
  • Find in Scripts: Enter a phrase and DS Game Maker will search Scripts in the project for this phrase.
  • Find Resource: Enter the name of a resource and DS Game Maker will open that resource. Use this if you have many resources in the project.
  • Last Found Resource: Opens the last resource found using the 'Find Resource' button.

Resources Menu

Use this menu to add the different types of resources to your game. You can also right click on a resource folder in the tree at the left of the program to see the appropriate command such as 'Add Room' or 'Add Sprite'.

Tools Menu

  • Game Settings: Edit settings for the game, such as the first room when the game starts, game description texts and default values for score variables.
  • Global Variables: Edit the global variables in the game. See 'Taking it further' > 'Global Variables'.
  • Global Arrays: Edit global arrays in the game. See 'Taking it further' > 'Arrays and Structures'.
  • Global Structures: Edit global structures in the game. See 'Taking it further' > 'Arrays and Structures'.
  • Action Editor: Opens the 'Action Editor' which you can use to edit all of the included Actions in the program (and also create your own).
  • Font Viewer: Opens a window for browsing the fonts you can use in games.
  • Run NO$GBA: Runs the included emulator (NO$GBA), you can then load an NDS file on your computer to play it.
  • Advanced:
    • Clean Up: Frees some disk space by removing redudant data created by DS Game Maker in the past
    • Edit Internal XDS: Edit the internal data for the game (use with caution as inappropriate use will lead to data loss)
    • Open Compile Temp: Opens the folder containing the last compiled game source code
    • Open Project Temp: Opens the folder containing the game data (use with caution)
    • Signifiy Graphics Change: Simulates a change in the game graphic data - clears the cache and forces each graphic to be reprocessed
    • Signify Graphics Change: Clears the sound cache so that all sounds are reprocessed when compiling.

Window Menu

Use this simple menu to switch between the windows open within DS Game Maker.

Help Menu

  • Contents: Open this help viewer.
  • Online Tutorials: Opens the 'Tutorials' section of the Forum.
  • Upgrade to Pro: Presents a diaog about upgrading to the Pro Edition.
  • Enter Serial Code: Have a Serial Code for the Pro Edition? Enter it by clicking this button.
  • News: Opens the 'News' window so you can see what is happening in the DS scene.
  • DSGameMaker.com: Opens the DSGameMaker.com website.
  • Discussion Forum: Opens the DS Game Maker Forum.
  • Run 'devkitPro Updater': Runs the automatic updater for the 'devkitPro' offered software. See 'Using DS Game Maker' > 'Install devkitARM & PAlib'.
  • About DS Game Maker: Opens a window showing the current version of the software.
