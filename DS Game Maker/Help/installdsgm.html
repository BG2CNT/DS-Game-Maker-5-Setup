Install DS Game Maker

The setup process for the software is fast and simple.

Microsoft .NET Framework

Before beginning, ensure that the 'Microsoft .NET Framework' is installed. Windows 7 users have this by default but Windows XP or Vista users may not have it, or have an outdated version. This is so if you receive the following error:

Click here to visit the Microsoft website to install the Framework.

Download and Run the Installer

DS Game Maker is available from the DSGameMaker.com website. The latest version will always be available from the website. Once you have downloaded the installer, run it. There are only a few steps to follow and there is no need to restart your computer after installation.

After Installation

Upon successful installation of DS Game Maker, you will see the below message:

Please see the next help document, 'Install devkitARM & PAlib'.
